# QT-Ind1

Реализовать класс "Множество на базе двусвязного списка":

Класс должен содержать следующие методы:
1) добавление элемента в множество;
2) исключение элемента из множества;
3) проверка множества на пустоту;
4) поиск элемента во множестве.

Для класса необходимо перегрузить следующие операторы:
1) operator|= добавление элемента;
2) operator/= исключение элемента;
3) operator|| объединение множеств:
4) operator&& пересечение множеств;
5) операторы сравнения множеств на равенство/ неравенство;
6) ввод/вывод в поток.

***************************************************

Используя множество строк решить задачу:
Даны два текстовых файла. Вывести все слова которые встречаются как в первом файле так и во втором документом.
